# 贷款管理

本应用可以将个人借贷与支出分开管理。您借给别人的钱记录为资产，您从别人那里借来的钱记录为负债。

如果将借贷当作支出处理，日后还款或收回时实际情况会变得难以看清。贷款管理让您可以分别查看未收回金额、未偿还金额和已完成的历史记录。

## 短期与长期

| 类型 | 用途 |
| --- | --- |
| 短期借出 | 临时垫付、小额借款给朋友等 |
| 长期借出 | 还款期较长的贷款，或希望按对方跟踪余额的贷款 |
| 短期借入 | 他人为您临时垫付的金额 |
| 长期借入 | 贷款、分期还款，或希望按对方跟踪余额的借款 |

短期借贷作为单个未结算交易进行管理，还款或收回时可以选择结算哪笔交易。长期借贷主要按账户余额管理。

## 如何输入

在输入页面，选择简易输入中的贷款/还款。有四种方向：

| 方向 | 含义 |
| --- | --- |
| 借入 | 您借了钱，或他人代您支付 |
| 还款 | 您偿还了所欠的钱 |
| 借出 | 您借出钱，或代他人支付 |
| 收回 | 您收回了他人欠您的钱 |

对于借入或借出，选择目标贷款账户和对方资产或支出账户。对于还款或收回，选择要结算的交易，以便应用知道哪笔贷款已被结清。

## 金额有差异的还款或收回

还款或收回的金额可能因手续费、折扣、汇兑差额、取整、赠与等原因与原始金额不完全一致。

在这种情况下，选择差额账户将差额记录为损益。例如，收到的金额超过借出金额是收入，少于借出金额是支出。

## 贷款管理页面

财务报表 -> 贷款管理列出借出和借入。页面分为短期借出、长期借出、短期借入和长期借入四个部分。

未结算的短期项目以卡片形式显示。已完成的短期项目和余额归零的长期项目存储在已完成的手风琴组件中。

从每个卡片中，按下"从此处输入"打开输入页面上的贷款输入流程。对于长期借出或借入，您也可以在实际上已结清但难以从账簿余额判断时，强制将其标记为已完成。

## 结清贷款

短期借出或借入在记录还款或收回分录后变为已完成。如果金额与原始金额完全匹配，选择目标交易并结算——这就是标记为完成所需做的全部。

长期借出或借入在余额达到零时自动视为完成。如果实际已还款但余额未归零，请使用贷款管理页面上的"强制完成"。但建议在强制完成之前先检查余额不匹配的原因（例如遗漏分录或输入错误）。

## 处理还款或收回中的差额

还款或收回的金额可能不与原始金额完全一致。

| 情况 | 处理方法 |
| --- | --- |
| 收回金额超过借出金额 | 将差额记录为收入（例如杂项收入、利息收入） |
| 收回金额少于借出金额 | 将差额记录为支出（例如杂项损失） |
| 还款金额少于借款金额（债务免除） | 将差额记录为收入（债务豁免收益） |
| 还款金额超过借款金额（利息或手续费） | 将差额记录为支出（利息支出、手续费） |

在每种情况下，目标都是使贷款账户余额归零。您可以在贷款输入表单中选择差额账户，无需手动构建多行分录。

## 处理坏账

当应收账款预计无法收回时，将其作为坏账支出核销。

步骤：
1. 在设置中，创建一个坏账支出账户（如果尚不存在）。
2. 使用多行分录，贷方记借出账户（减少资产），借方记坏账账户（增加支出）。
3. 在贷款管理页面上，使用强制完成将该项目移至已完成。

如果只能收回部分金额，将实际收到的现金分录与核销剩余部分的坏账分录结合使用。

## 如何创建账户

如果您有许多临时垫付，使用共享账户如短期借出和短期借入通常更方便。如果您希望在长期内跟踪特定人员或贷款余额，请创建单独的长期账户，例如"借给A"或"住房贷款"。

借出和借入本身不是家庭支出。资金流入或流出，但由于预计将归还或需要偿还，因此作为资产或负债处理。如果最终不会归还或无需再偿还，则作为损益或杂项损益处理。
