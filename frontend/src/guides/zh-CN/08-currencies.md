# 多币种处理

本应用支持多种货币。您可以为每个账户设置币种，因此外币银行账户和外币资产可以纳入账簿。

## 核心思路

每笔分录按币种记录。例如，将1,000美元存入美元银行账户以美元记录。应用不会自动获取汇率。记录外币交易时，直接输入外币金额。要在财务报表中查看换算后的总额，请手动输入换算分录。

## 创建外币账户

1. 在设置中，创建一个新账户。
2. 将账户币种设置目标币种，如USD或EUR。
3. 使用期初余额表单以外币输入期初余额。

## 记录外币交易

外币存取款与普通条目输入方式相同。以外币输入金额。

| 交易示例 | 输入方法 |
| --- | --- |
| 向美元账户存入1,000美元 | 选择美元账户作为收入目标，输入1,000 |
| 从美元账户支付100美元支出 | 选择美元账户作为支出来源，输入100 |
| 将美元账户转换为日元账户 | 在两个账户之间使用多行分录 |

## 币种换算分录

在日元账户和外币账户之间进行换算时，使用多行分录。

**示例：将1,000美元（按150日元/美元购入）兑换为145,000日元**

| 借方 | 贷方 |
| --- | --- |
| 银行账户（日元）145,000 | 美元账户 1,000（成本基础等值150,000日元） |
| 汇兑损失 5,000 | |

购入汇率与兑换汇率之间的差额成为汇兑损益，记录为收入或支出。

本应用支持允许借方/贷方金额不匹配的币种兑换分录（`is_currency_exchange`）。在多行分录表单中，将分录记录为币种兑换可以保存不同币种金额的组合。

## 在财务报表中的显示

在财务报表汇总中，不同币种的余额以纯数字形式并排显示。例如，如果您在日元账户中有100,000日元，在美元账户中有500美元，资产总计显示为100,000 + 500的组合数字。

当币种混合时，请将总计仅作为参考值。为获得准确的总计，请手动检查换算后的余额，或通过输入换算分录合并为单一币种。

## 更改币种

已有分录的账户币种无法更改。如果币种设置错误，请使用批量编辑与替换功能将分录移至另一个账户，然后删除原账户。

添加新币种（例如开设新的欧元账户）只需创建一个设置为EUR的新账户。现有账户和分录不受影响。

## 加密资产与币种

BTC、ETH、SOL等加密资产也被视为一种外币类型。创建一个加密类别的账户并将币种设置为BTC或类似币种。然后分录以该加密单位记录。

要查看日元等值价值，请使用加密资产监控从外部源获取价格并乘以持有数量。详情请参阅加密资产监控指南。
