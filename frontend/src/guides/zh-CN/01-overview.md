# 关于本应用

本应用是一款围绕个人资产负债表管理个人财务的家庭理财工具，而不仅限于日常支出记录。它不仅显示"这个月花了多少钱"，而是将现金、银行存款、信用卡、借贷、加密资产、固定资产和预算余额作为一套账簿进行管理。

应用内部采用复式记账法。但在日常输入时，您无需考虑借贷方向。简易输入让您选择支出、收入、转账或借贷相关交易，应用会在后台自动创建所需的分录。

## 本应用管理的领域

| 领域 | 主要页面 | 功能 |
| --- | --- | --- |
| 日常输入 | 输入 | 记录支出、收入、转账、借贷及CSV导入 |
| 账簿查看 | 账簿 | 以简易视图或复式视图查看分录 |
| 财务状况 | 财务报表 | 查看资产、负债、净资产、损益及财务健康度 |
| 预算管理 | 概览、预算设置 | 在现金和存款之上创建基于用途的虚拟桶，管理可用余额 |
| 对账 | 财务报表 -> 试算表 | 比较实际余额与账面余额，核对信用卡或不明资金 |
| 特殊资产 | 加密资产、储蓄、贷款管理 | 管理钱包余额、储蓄目标、借出和借入 |

## 基本概念

在本应用中，您首先需要创建账户。账户是直接出现在账簿中的实际项目，例如银行账户、信用卡、工资、餐饮支出和应收贷款。每个账户有五种类型之一：资产、负债、净资产、收入或支出。

预算类别与账户不同。预算类别是叠加在现金和银行存款等实际资产之上的虚拟用途桶。例如，如果您的银行余额为10万日元，您可以将3万日元视为餐饮、2万日元视为娱乐、5万日元视为储蓄。

## 初始设置步骤

1. 在设置中，创建您使用的银行账户、信用卡、收入账户和主要支出账户。
2. 在初始余额输入中，登记您起始点的账户和卡余额。
3. 在预算设置中，创建餐饮、娱乐、储蓄等预算类别。
4. 如有需要，创建可自动将收入分配到各预算类别的预算过滤器。
5. 从输入页面记录日常交易。
6. 偶尔使用试算表页面检查实际余额是否与账簿一致。

## 如何选择页面

日常使用中，概览和输入页面可能就足够了。检查预算时使用设置 -> 预算设置，余额不匹配时使用财务报表 -> 试算表，需要仔细查看信用卡、贷款或其他细节时使用财务报表下的子页面。

如果您了解复式记账法，多行分录输入和账簿的复式视图可提供更精细的控制。如果您不了解，本应用的设计也能让您通过简易输入和试算表检查完成日常管理。
