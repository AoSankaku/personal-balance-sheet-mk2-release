# 模拟折旧与预算分摊

本应用中的折旧并非用于税务申报或正式公司会计。它是一项家庭理财功能，用于将大额购物的负担分摊到多个月份。

例如，如果您在一月份购买了一台12万日元的电器，普通的家庭账簿会显示一月支出很大。但实际操作中，您可能希望将其视为一年内每月消费1万日元。模拟折旧正是为此场景设计。

## 实现方式

在输入支出时启用折旧，应用会创建两种类型的记录：

| 记录 | 内容 |
| --- | --- |
| 购买记录 | 减少支付资产，增加可折旧资产 |
| 月度记录 | 每月逐步减少资产并记录支出 |

现金在购买时减少，但对支出和预算的影响按月分摊。在资产负债表上，尚未费用化的部分仍作为资产保留。

## 使用前设置

1. 在设置中，创建您要折旧的资产账户。
2. 为该资产账户启用折旧。
3. 准备一个用于折旧支出的支出账户。
4. 在输入页面输入支出时，启用折旧。

在基于期间输入时，指定总金额应分摊到多少个月。在月度金额输入时，指定每月应费用化的金额。如果需要取整，月度金额会进行调整以确保总额仍然匹配。

## 对预算的影响

对于普通支出，预算在付款当月消耗。对于折旧，预算影响表现为月度折旧支出。

这使得可以将大额年度采购或可用数月的用品作为月度生活成本的一部分进行平滑处理。目标是在支付月份与使用期间之间进行分离，使其与家庭实际感受的成本相匹配。

## 适合的支出

| 适合的项目 | 原因 |
| --- | --- |
| 电器、家具、电脑、智能手机 | 可使用数月至数年 |
| 年度订阅 | 实际上每月都在使用 |
| 昂贵的爱好或工作设备 | 避免仅在购买月扭曲预算 |
| 改善生活品质的一次性支出 | 收益跨越数月 |

对于餐饮、日用品和其他当月消耗的物品，请使用普通支出输入。

## 编辑和删除注意事项

当您编辑创建折旧的购买分录时，月度折旧分录会重新计算。仅编辑某一月度分录可能导致其与整体计划不一致。

如果需要更改详情，请尽可能从原始购买分录编辑。如果不再需要，删除相关的折旧计划，必要时重新输入为普通支出。

## 与税务折旧的区别

此功能是用于家庭管理的模拟费用分摊。它不会自动确定使用年限、折旧方法、业务使用比例或税务处理。需要税务处理的资产应根据正式账簿和税务规则单独管理。
