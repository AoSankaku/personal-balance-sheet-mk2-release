# このアプリについて

このアプリは、日々の支出入力だけでなく、個人の貸借対照表を中心にお金の状態を管理するための家計簿です。一般的な家計簿のように「今月いくら使ったか」を見るだけではなく、現金、預金、クレジットカード、貸付、借入、暗号資産、固定資産、予算の残りをひとつの帳簿として扱います。

内部では複式簿記の仕組みを使っています。ただし、普段の入力では借方・貸方を意識しなくてもよいように、シンプル入力が用意されています。支出、収入、振替、貸し借りを選んで入力すると、アプリが裏側で必要な仕訳を作成します。

## このアプリで管理するもの

| 領域 | 主なページ | できること |
| --- | --- | --- |
| 日々の入力 | 入力 | 支出、収入、振替、貸し借り、CSVインポートを記録する |
| 帳簿確認 | 帳簿 | 入力された仕訳をシンプル表示または複式簿記表示で確認する |
| 財務状態 | 財務諸表 | 資産、負債、純資産、損益、財政健全度を見る |
| 予算 | 概要、予算設定 | 現金・預金の上に目的別の仮想バケットを作り、使える残高を管理する |
| 試算・照合 | 財務諸表 -> 試算表 | 実際の残高と帳簿上の残高を照合し、クレジットカードや不明金を確認する |
| 特殊資産 | 暗号資産、貯蓄、ローン管理 | ウォレット残高、貯蓄目標、貸付・借入の状態を管理する |

## 基本の考え方

このアプリでは、まず「勘定科目」を作ります。勘定科目は、銀行口座、クレジットカード、給与、食費、貸付金など、帳簿に直接登場する実体です。勘定科目には、資産、負債、資本、収益、費用の5種類があります。

一方で「予算カテゴリー」は、勘定科目ではありません。予算カテゴリーは、現金・預金などの実際の資産の上に乗る仮想的な目的別バケツです。たとえば銀行口座の残高が100,000円あるとして、そのうち30,000円を食費、20,000円を娯楽費、50,000円を貯蓄として見る、という使い方をします。

## 最初にやること

1. 設定で、使っている銀行口座、クレジットカード、収入、主な費用科目を作ります。
2. 初期残高入力で、開始時点の口座残高やカード残高を登録します。
3. 予算設定で、食費、娯楽費、貯蓄などの予算カテゴリーを作ります。
4. 必要に応じて、収入を各予算カテゴリーに自動配分する予算フィルターを作ります。
5. 入力ページから日々の取引を記録します。
6. ときどき試算表で、実際の残高と帳簿が合っているか確認します。

## 使い分けの目安

普段は「概要」と「入力」だけでも使えます。予算の見直しをするときは「設定 -> 予算設定」、残高が合わないときは「財務諸表 -> 試算表」、カードや借入の状態を詳しく見るときは「財務諸表」の各サブページを使います。

複式簿記を理解している場合は、複合仕訳や帳簿の複式簿記表示を使うと、より細かい調整ができます。理解していない場合でも、シンプル入力と試算表を中心に使えば、日常的な管理はできるように設計されています。
