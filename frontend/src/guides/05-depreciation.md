# 擬似的な減価償却と予算分散

このアプリの減価償却は、税務申告や法人会計の正式な減価償却を目的としたものではありません。大きな支出を、家計管理上の負担として複数月に分散して見るための機能です。

たとえば、120,000円の家電を1月に買った場合、通常の家計簿では1月だけ大きな支出になります。しかし実感としては、1年間使うものを毎月10,000円ずつ消費していると見たい場合があります。このようなときに、擬似的な減価償却を使います。

## 何が起きるか

減価償却を有効にして支出を入力すると、アプリは次の2種類の記録を作ります。

| 記録 | 内容 |
| --- | --- |
| 購入時の記録 | 支払元の資産を減らし、減価償却対象の資産を増やす |
| 月次の記録 | 対象資産を少しずつ減らし、費用を毎月発生させる |

これにより、現金は購入時に減りますが、費用や予算への影響は月ごとに分散されます。貸借対照表では、まだ費用化されていない部分が資産として残ります。

## 使う前の準備

1. 設定で、減価償却対象にしたい資産科目を作ります。
2. その資産科目で「減価償却対象」を有効にします。
3. 減価償却費として使う費用科目を用意します。
4. 入力ページで支出を入力するとき、減価償却を有効にします。

期間入力では、総額を何ヶ月に分けるかを指定します。月額入力では、毎月いくらずつ費用化するかを指定します。端数が出る場合は、全体が合うように月次金額が調整されます。

## 予算への影響

通常の支出では、支出した月に予算が消費されます。減価償却を使うと、毎月の減価償却費として予算に反映されます。

これにより、年に1回だけ発生する大きな買い物や、数ヶ月使う備品を、月次の生活費としてならして管理できます。家計の実感に合わせて、支出の発生月と利用期間を分けられる点がこの機能の目的です。

## 向いている支出

| 向いているもの | 理由 |
| --- | --- |
| 家電、家具、PC、スマートフォン | 数ヶ月から数年かけて使うため |
| 年払いのサービス | 実質的には毎月利用しているため |
| 趣味や仕事用の高額備品 | 購入月だけ予算が大きく崩れるのを避けられるため |
| 生活改善のための一時支出 | 効果が複数月にまたがるため |

食費や日用品のように、その月で消費されるものには通常の支出入力を使います。

## 編集と削除の注意

減価償却の元になった購入仕訳を編集すると、月次の減価償却仕訳も再計算されます。月次仕訳だけを個別に編集すると、スケジュール全体の計算と合わなくなる場合があります。

内容を変更したい場合は、できるだけ元の購入仕訳から編集してください。不要になった場合は、関連する減価償却スケジュールを削除し、必要に応じて通常の支出として入力し直します。

## 税務上の減価償却との違い

この機能は、家計管理のための擬似的な費用分散です。耐用年数、償却方法、事業割合、税務申告上の扱いを自動で判断するものではありません。税務上の処理が必要な資産は、別途正式な帳簿や税務ルールに従って管理してください。
