# 多幣種處理

此應用程式支援多種貨幣。您可以為每個帳戶設定貨幣，因此外幣銀行帳戶與外幣資產都可以納入帳簿。

## 核心概念

每筆分錄按貨幣記錄。例如，將 1,000 美元存入美元銀行帳戶會以美元記錄。應用程式不會自動擷取匯率。記錄外幣交易時，直接輸入外幣金額。若要在財務報表中檢視換算後的合計金額，請手動輸入換算分錄。

## 建立外幣帳戶

1. 在設定中，建立一個新帳戶。
2. 將帳戶貨幣設定為目標貨幣，例如 USD 或 EUR。
3. 使用期初餘額表單以外幣輸入期初餘額。

## 記錄外幣交易

外幣存款與提款的輸入方式與一般分錄相同。以外幣輸入金額。

| 交易範例 | 輸入方式 |
| --- | --- |
| 將 1,000 美元存入美元帳戶 | 選擇美元帳戶作為收入目標，輸入 1,000 |
| 從美元帳戶支付 100 美元的支出 | 選擇美元帳戶作為支出來源，輸入 100 |
| 將美元帳戶轉換為日圓帳戶 | 在兩個帳戶之間使用多行分錄 |

## 貨幣換算分錄

在日圓帳戶與外幣帳戶之間換算時，請使用多行分錄。

**範例：將 1,000 美元（以 150 日圓／美元取得）換算為 145,000 日圓**

| 借方 | 貸方 |
| --- | --- |
| 銀行帳戶（日圓）145,000 | 美元帳戶 1,000（成本基礎 150,000 日圓等值） |
| 外匯損失 5,000 | |

取得匯率與換算匯率之間的差額成為外匯損益，並記錄為收入或支出。

此應用程式支援允許借方／貸方金額不匹配的貨幣兌換分錄（`is_currency_exchange`）。在多行分錄表單中，將分錄記錄為貨幣兌換可以讓您儲存不同貨幣金額的組合。

## 在財務報表中的顯示

在財務報表合計中，不同貨幣的餘額會以單純的數字並列顯示。例如，如果您在日圓帳戶中有 100,000 日圓，在美元帳戶中有 500 美元，資產總計會顯示為 100,000 + 500 的合併數字。

當貨幣混合時，請將合計值僅作為參考值。如需準確的總計，請手動檢查換算後的餘額，或透過輸入換算分錄將所有資產合併為單一貨幣。

## 變更貨幣

已有分錄的帳戶無法變更貨幣。如果貨幣設定錯誤，請使用批次編輯與取代功能將分錄移至其他帳戶，然後刪除原始帳戶。

新增貨幣（例如開設新的歐元帳戶）只需建立一個設定為 EUR 的新帳戶。現有帳戶與分錄不受影響。

## 加密資產與貨幣

加密資產（例如 BTC、ETH 及 SOL）也被視為一種外幣。在加密貨幣類別中建立帳戶，並將貨幣設定為 BTC 或類似單位。分錄隨後以該加密貨幣單位記錄。

若要檢查日圓等值金額，請使用加密資產監看從外部來源擷取價格並乘以持有數量。詳細資訊請參閱加密資產監看指南。
