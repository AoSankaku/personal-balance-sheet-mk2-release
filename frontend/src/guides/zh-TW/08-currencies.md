# 多幣種處理

此應用程式可以在貨幣設定中啟用多種貨幣，並在它們之間切換。貨幣會記錄在每一筆分錄明細上，因此可以按貨幣分別管理外幣與加密資產餘額。

## 核心概念

畫面上方的貨幣切換器會決定輸入與顯示使用的貨幣。例如，選擇 USD 後在美元銀行帳戶中記錄 1,000 的存款，該分錄會以 USD 儲存。

帳戶並不限定為單一貨幣。您既可以在同一帳戶中保留多種貨幣餘額，也可以依貨幣與用途建立不同帳戶。需要明確分開餘額時，可使用「USD 銀行帳戶」「BTC－日常」等專用帳戶。

## 開始使用外幣

1. 在貨幣設定中啟用 USD、EUR 等目標貨幣。
2. 透過畫面上方的貨幣切換器選擇該貨幣。
3. 如果希望依用途分開餘額，請建立專用帳戶。
4. 使用期初餘額表單，以目前選擇的貨幣記錄餘額。

## 記錄外幣交易

外幣存款與提款的輸入方式與一般分錄相同。以外幣輸入金額。

| 交易範例 | 輸入方式 |
| --- | --- |
| 將 1,000 美元存入美元帳戶 | 選擇美元帳戶作為收入目標，輸入 1,000 |
| 從美元帳戶支付 100 美元的支出 | 選擇美元帳戶作為支出來源，輸入 100 |
| 將美元帳戶轉換為日圓帳戶 | 在兩個帳戶之間使用多行分錄 |

## 貨幣換算分錄

在日圓帳戶與外幣帳戶之間換算時，請使用多行分錄。

**範例：將 1,000 美元（以 150 日圓／美元取得）換算為 145,000 日圓**

| 借方 | 貸方 |
| --- | --- |
| 銀行帳戶（日圓）145,000 | 美元帳戶 1,000（成本基礎 150,000 日圓等值） |
| 外匯損失 5,000 | |

取得匯率與換算匯率之間的差額成為外匯損益，並記錄為收入或支出。

此應用程式支援允許借方／貸方金額不匹配的貨幣兌換分錄（`is_currency_exchange`）。在多行分錄表單中，將分錄記錄為貨幣兌換可以讓您儲存不同貨幣金額的組合。

## 在財務報表中的顯示

財務報表通常只顯示貨幣切換器目前所選貨幣的餘額。啟用「包含所有貨幣」後，其他貨幣餘額會依貨幣設定中的匯率換算為顯示貨幣，並計入合計。

換算值會隨價格來源與擷取時間而改變。帳簿仍保留原貨幣數量；顯示換算不會修改分錄。

## 新增或移除貨幣

新增貨幣不會影響現有帳戶或分錄。仍有餘額的貨幣無法停用；不再需要時，請先透過轉帳或貨幣兌換分錄將餘額歸零。

## 加密資產與貨幣

加密資產支援兩種使用模式。

- 用於日常支付時，請在貨幣設定中啟用 BTC 等加密貨幣，透過貨幣切換器選擇後記錄日常收入、支出與轉帳。如需分開管理餘額，可建立「BTC－日常」等現金帳戶。
- 用於投資或投機時，請在加密資產類別中建立「BTC－投資」等帳戶，將買入、賣出、獎勵與手續費記錄為投資活動。

如果家庭財務以日圓為基準，就不需要為加密資產價格變動建立分錄。購買時記錄支付的日圓與取得數量，出售時記錄收到的日圓與處分數量，直接使用加密資產付款時記錄支出金額與處分數量。貨幣換算顯示只是參考資訊，不會修改帳簿。

同一 BTC 可以同時用於兩種目的。即使貨幣代碼相同，不同帳戶的餘額仍會保持獨立，餘額核對也會逐一比較每個帳戶。只有在頁首選擇 BTC 期間，才可將錢包數量擷取到投資帳戶；包括日常使用帳戶在內的所有帳戶都可以手動輸入實際餘額。詳細資訊請參閱加密資產餘額核對指南。
