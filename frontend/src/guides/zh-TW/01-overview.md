# 關於此應用程式

此應用程式是一款家計財務工具，以個人資產負債表為核心管理您的金錢，而不僅限於日常支出記錄。它不僅顯示「這個月花了多少錢？」，而是將現金、銀行存款、信用卡、借貸、加密資產、固定資產以及剩餘預算視為一套帳簿進行管理。

在內部，此應用程式採用複式簿記。但對於日常輸入，您無需思考借貸方向。簡易輸入讓您選擇支出、收入、轉帳或借貸相關交易，應用程式會在幕後自動建立所需的分錄。

## 此應用程式管理哪些項目

| 項目 | 主要頁面 | 您可以做的事 |
| --- | --- | --- |
| 日常輸入 | 輸入 | 記錄支出、收入、轉帳、借貸以及 CSV 匯入 |
| 帳簿檢視 | 帳簿 | 以簡易檢視或複式檢視查看分錄 |
| 財務狀態 | 財務報表 | 查看資產、負債、淨值、損益及財務健康狀況 |
| 預算管理 | 總覽、預算設定 | 在現金與存款之上建立以用途為基礎的虛擬桶，並管理可用餘額 |
| 對帳 | 財務報表 -> 試算表 | 比對實際餘額與帳面餘額，並檢查信用卡或不明資金 |
| 特殊資產 | 加密貨幣、儲蓄、貸款管理 | 管理錢包餘額、儲蓄目標、借貸 |

## 基本概念

在此應用程式中，您首先需要建立帳戶。帳戶是直接出現在帳簿中的實際項目，例如銀行帳戶、信用卡、薪資、餐飲支出及應收貸款。每個帳戶屬於五種型別之一：資產、負債、權益、收入或支出。

預算類別與帳戶不同。預算類別是建立在現金與銀行存款等實際資產之上的虛擬用途桶。例如，如果您的銀行餘額為 100,000 日圓，您可以將 30,000 日圓視為餐飲、20,000 日圓視為娛樂、50,000 日圓視為儲蓄。

## 首次設定步驟

1. 在設定中，建立您使用的銀行帳戶、信用卡、收入帳戶及主要支出帳戶。
2. 在初期餘額輸入中，登錄您起始點的帳戶與卡片餘額。
3. 在預算設定中，建立餐飲、娛樂、儲蓄等預算類別。
4. 如有需要，建立可自動將收入分配至各預算類別的預算篩選器。
5. 從輸入頁面記錄每日交易。
6. 偶爾使用試算表頁面檢查實際餘額是否與帳簿相符。

## 如何選擇頁面

日常使用中，總覽與輸入頁面可能已足夠。檢視預算時使用設定 -> 預算設定，餘額不符時使用財務報表 -> 試算表，想仔細查看卡片、貸款或其他細節時使用財務報表下的子頁面。

如果您瞭解複式簿記，多行分錄表單與帳簿的複式檢視能提供更精細的控制。如果您不熟悉，此應用程式的設計讓日常管理仍可透過簡易輸入與試算表檢查來運作。
