# 模擬折舊與預算分攤

此應用程式中的折舊並非用於報稅或正式公司會計。它是一項家計財務功能，用於將大額採購的負擔分攤到多個月份。

例如，如果您在一月份購買了一臺 120,000 日圓的家電，普通的家計帳簿會顯示一月份有一筆大額支出。實際上，您可能希望將其視為一年內每月消耗 10,000 日圓。模擬折舊就是為此情況而設。

## 運作方式

當您在輸入支出時啟用折舊，應用程式會建立兩類記錄：

| 記錄 | 內容 |
| --- | --- |
| 購買記錄 | 減少付款資產並增加可折舊資產 |
| 每月記錄 | 每月逐步減少資產並記錄支出 |

現金在購買時減少，但對支出與預算的影響則按月分攤。在資產負債表上，尚未費用化的部分仍作為資產留存。

## 使用前設定

1. 在設定中，建立您想要折舊的資產帳戶。
2. 為該資產帳戶啟用折舊。
3. 準備一個用於折舊費用的支出帳戶。
4. 在輸入頁面上輸入支出時，啟用折舊。

在期間制輸入中，指定總額要分攤多少個月。在月額制輸入中，指定每個月應費用化的金額。如果需要四捨五入，每月金額會調整以使總額仍然相符。

## 對預算的影響

一般支出會在付款當月消耗預算。使用折舊時，預算影響會以每月折舊費用的形式出現。

這使得平滑處理大額年度採購或可使用數月的用品，使其成為每月生活成本的一部分成為可能。目標是將付款月份與使用期間分開，以符合家庭實際感受成本的方式處理。

## 適合的支出

| 適合項目 | 原因 |
| --- | --- |
| 家電、傢俱、個人電腦、智慧型手機 | 使用數月或數年 |
| 年度訂閱 | 實際上每月都在使用 |
| 昂貴的興趣或工作裝置 | 避免預算僅在購買月份失真 |
| 改善生活品質的一次性支出 | 效益涵蓋多個月 |

對於餐飲、日用品及其他當月消耗完畢的物品，請使用一般支出輸入。

## 編輯與刪除注意事項

當您編輯建立折舊的原始購買分錄時，每月折舊分錄會重新計算。僅編輯某個月份的分錄可能會導致與整體排程不一致。

如果需要變更明細，請盡可能從原始購買分錄編輯。如果不再需要，請刪除相關的折舊排程，並在必要時重新以一般支出輸入。

## 與稅務折舊的差異

此功能是供家計管理使用的模擬費用分攤。它不會自動決定耐用年限、折舊方法、業務使用比例或稅務處理。需要稅務處理的資產應根據正式帳簿與稅務規則另行管理。
