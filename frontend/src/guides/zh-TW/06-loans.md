# 貸款管理

此應用程式可以將個人借貸與貸款與支出分開管理。您借給他人的錢記錄為資產，您從他人借入的錢記錄為負債。

如果將借貸視為支出，日後還款或收回時將難以看清實際情況。貸款管理讓您可以分別檢視未收回金額、未償還金額及已完成記錄。

## 短期 vs 長期

| 型別 | 用途 |
| --- | --- |
| 短期貸出 | 臨時墊款、小額借款給朋友等 |
| 長期貸出 | 還款期長的貸款，或希望按對方追蹤餘額的貸款 |
| 短期借入 | 他人暫時代付的金額 |
| 長期借入 | 貸款、分期還款，或希望按對方追蹤餘額的借款 |

短期借貸以個別未結算交易的方式管理，您可以在還款或收回時選擇結算哪筆交易。長期借貸主要以帳戶餘額方式管理。

## 輸入方式

在輸入頁面上，選擇簡易輸入中的貸款／還款。有四種方向：

| 方向 | 含義 |
| --- | --- |
| 借入 | 您借了錢，或他人代您付款 |
| 還款 | 您償還了所欠款項 |
| 貸出 | 您借出錢，或代他人付款 |
| 收回 | 您收回了他人欠您的款項 |

對於借入或貸出，請選擇目標貸款帳戶及對方資產或支出帳戶。對於還款或收回，請選擇要結算的交易，以便應用程式知道哪筆貸款已被清償。

## 有差額的還款或收回

還款或收回的金額可能因手續費、折扣、匯兌差額、四捨五入、贈與等原因而與原始金額不完全相符。

在這種情況下，請選擇一個差額帳戶將差額記錄為收益或損失。例如，收到的金額超過貸出的金額屬於收入，收到的金額少於貸出的金額則屬於支出。

## 貸款管理頁面

財務報表 -> 貸款管理列出了貸出與借入。頁面分為短期貸出、長期貸出、短期借入與長期借入。

未結算的短期項目以卡片形式顯示。已完成的短期項目及餘額歸零的長期項目儲存在已完成的摺疊區域中。

從每張卡片按下「從此處輸入」會開啟輸入頁面上的貸款輸入流程。對於長期貸出或借入，您也可以強制將某項目標記為已完成，用於實際上已完成但難以從帳面餘額判斷的情況。

## 結清貸款

短期貸出或借入在記錄還款或收回分錄後即變為已完成。如果金額與原始金額完全相符，選擇目標交易並結算即可標記為完成。

長期貸出或借入在餘額歸零時自動視為完成。如果還款實際上已完成但餘額未歸零，請使用貸款管理頁面上的「強制完成」。不過，建議在強制完成之前先檢查餘額不符的原因（例如遺漏分錄或輸入錯誤）。

## 處理還款或收回的差額

還款或收回的金額可能與原始金額不完全相符。

| 情況 | 處理方式 |
| --- | --- |
| 收到金額多於貸出金額 | 將差額記錄為收入（例如雜項收入、利息收入） |
| 收到金額少於貸出金額 | 將差額記錄為支出（例如雜項損失） |
| 還款少於借款（債務免除） | 將差額記錄為收入（債務免除收益） |
| 還款多於借款（利息或手續費） | 將差額記錄為支出（利息支出、手續費） |

在所有情況下，目標都是使貸款帳戶餘額歸零。您可以在貸款輸入表單中選擇差額帳戶，無需手動建立多行分錄。

## 處理呆帳

當應收款項預期無法收回時，將其沖銷為呆帳費用。

步驟：
1. 在設定中，如果尚不存在，建立一個呆帳費用帳戶。
2. 使用多行分錄貸記貸出帳戶（減少資產）並借記呆帳帳戶（增加支出）。
3. 在貸款管理頁面上，使用強制完成將該項目移至已完成。

如果僅能收回部分金額，將一筆實際收到的現金分錄與另一筆沖銷剩餘呆帳的分錄結合使用。

## 如何建立帳戶

如果您有許多臨時墊款，使用共用帳戶（例如短期貸出與短期借入）通常較為方便。如果您希望長期追蹤特定人士或貸款餘額，請建立個別的長期帳戶，例如「貸給 A」或「房屋貸款」。

貸出與借入本身並非家計支出。資金進出，但因為預期會歸還或必須償還，所以視為資產或負債。如果最終不會歸還或無需再償還，則將其作為損益或雜項損益沖銷。
