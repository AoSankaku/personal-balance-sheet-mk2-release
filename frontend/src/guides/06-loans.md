# 貸し借り管理

このアプリでは、個人間の貸し借りやローンを、支出とは別に管理できます。誰かにお金を貸した場合は資産、誰かから借りた場合は負債として記録されます。

貸し借りを支出として処理してしまうと、後で返済や回収があったときに実態が見えにくくなります。貸し借り管理を使うと、未回収額、未返済額、完了済みの履歴を分けて確認できます。

## 短期と長期の違い

| 種類 | 使い方 |
| --- | --- |
| 短期貸付 | 一時的な立て替え、友人への少額貸付など |
| 長期貸付 | 返済期間が長い貸付、相手ごとに残高を追いたい貸付 |
| 短期借入 | 一時的に立て替えてもらった金額など |
| 長期借入 | ローン、分割返済、相手ごとに残高を追いたい借入 |

短期の貸し借りは、個別の発生取引を未解決として管理し、返済・回収時にどの取引を精算するか選べます。長期の貸し借りは、勘定科目ごとの残高を中心に管理します。

## 入力方法

入力ページのシンプル入力で「貸し借り」を選びます。方向は次の4つです。

| 方向 | 意味 |
| --- | --- |
| 借入 | 借りた、または立て替えてもらった |
| 返済 | 借りていた分を返した |
| 貸付 | 貸した、または立て替えた |
| 回収 | 貸していた分を受け取った |

借入や貸付では、対象となる貸借科目と、相手となる資産または費用科目を選びます。返済や回収では、精算対象の取引を選ぶことで、どの貸し借りが解決したかを記録できます。

## 差額がある返済・回収

返済額や回収額が元の金額と完全に一致しないことがあります。手数料、割引、為替差、端数、謝礼などがある場合です。

この場合、差額科目を選ぶことで、差益または差損として処理できます。たとえば、貸した金額より多く返ってきた場合は収益、少なく返ってきた場合は費用として扱います。

## ローン管理ページ

財務諸表 -> ローン管理では、貸付と借入を一覧できます。ページは、短期貸付、長期貸付、短期借入、長期借入に分かれています。

未解決の短期貸し借りはカードとして表示されます。完了した短期貸し借りや、残高が0になった長期貸し借りは、完了済みのアコーディオンに収納されます。

各カードから「ここから入力」を押すと、入力ページの貸し借り入力へ移動できます。長期貸付や長期借入では、実際には完了しているが帳簿上の判定が難しい場合に、強制完了として扱うこともできます。

## 貸し借りの完了処理

短期貸し借りは、返済・回収を記録することで「完了済み」に移ります。返済額が元の金額と完全に一致している場合は、対象の取引を選んで精算するだけで完了します。

長期貸し借りは、残高が0になると自動的に完了扱いになります。実際には返済・回収が終わっているが帳簿上の残高が0にならない場合は、ローン管理ページから「強制的に完了とする」を使うことができます。ただし強制完了を使う前に、不一致の原因（仕訳ミス、未入力の返済など）を確認することを推奨します。

## 差損・差益がある場合の処理

返済額や回収額が元の金額と異なる場合があります。

| ケース | 処理の考え方 |
| --- | --- |
| 貸した金額より多く戻ってきた | 差額を収益（雑収入、利息収入など）として記録 |
| 貸した金額より少なく戻ってきた | 差額を費用（雑損失など）として記録 |
| 借りた金額より少なく返した（免除された） | 差額を収益（債務免除益）として記録 |
| 借りた金額より多く返した（利息・手数料） | 差額を費用（利息費用、手数料など）として記録 |

いずれの場合も、貸し借り科目の残高は0になるように整理するのが原則です。貸し借り入力の「差額科目」欄で選択することで、複合仕訳を手動で作らずに処理できます。

## 貸倒の処理

回収できる見込みがなくなった貸付金は、「貸倒損失」として費用に振り替えます。

処理の手順：
1. 設定で「貸倒損失」の費用科目を用意します（まだない場合）。
2. 複合仕訳を使い、「貸付金（資産）を貸方」、「貸倒損失（費用）を借方」に記録します。
3. ローン管理ページで強制完了を使い、該当の貸付を完了済みに移します。

回収の一部だけ可能な場合は、回収できた分だけ現金として受け取る仕訳と、残りを貸倒損失に振り替える仕訳を組み合わせます。

## 勘定科目を作るときの考え方

短期の立て替えが多い場合は、まとめて「短期貸付」「短期借入」の科目を使うと管理しやすくなります。相手ごとに残高を長く追いたい場合は、「Aさん貸付」「住宅ローン」など、長期貸付・長期借入の科目を個別に作る方が向いています。

貸し借りは、家計の支出そのものではありません。お金の出入りはありますが、返ってくる予定や返す義務があるため、資産または負債として扱います。最終的に返ってこない、または返さなくてよくなった場合は、差損・差益や雑損・雑益として整理します。
