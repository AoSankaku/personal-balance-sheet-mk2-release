# 試算表とクレジットカード残高確認

試算表は、帳簿上の残高と実際の残高を照合するページです。財務諸表 -> 試算表から開きます。

## できること

| タブ | 目的 |
| --- | --- |
| 現在値入力 | 実際の口座残高やカード利用額を入力する |
| 誤差検証 | 現在値と帳簿残高を比較する |
| 不明金処理 | 原因不明の差異を一時的に処理する |
| 予算整合性チェック | 仕訳と予算配分、予算配置のずれを確認する |

## 口座とカードの照合

現在値入力で銀行口座、現金、負債、カード利用額を保存すると、誤差検証で帳簿価額、実際の値、差異を確認できます。差異がある場合は、同額の仕訳候補、最近の入力、日付違い、二重入力、返金、振替方向の誤りを確認します。

クレジットカードは、利用日、締め日、確定日、引落日、請求月が分かれるためずれやすい科目です。カード明細が確定したら、該当する支払い月の金額を現在値入力に保存し、帳簿上の利用額と比較します。

## 予算整合性チェック

予算整合性チェックでは、支出・収入の仕訳額と保存された予算配分額が一致しているかを確認します。費用科目に予算配分があるのに仕訳側に配分がない場合や、金額がずれている場合に検出されます。

複合仕訳や予算配分対象外の取引では、明示的な予算配分がない限り、アプリは予算カテゴリを推測して適用しません。必要に応じて帳簿から仕訳を編集するか、予算調整で現在の予算だけを直します。

このタブには予算配置表も表示されます。予算カテゴリに設定した目標保管口座と実際の現金・預金残高を比較し、どの口座グループへいくら移動するか、または予算を追加・減額するかのヒントを確認できます。
